How to HTML edit

If for some random reason you don't already know

Steps

  1. Open developer tools
    You will see the DOM open. This step can be achieved by the F12 button or right click and inspect.
  2. Find what you want to edit
    Go to the "Elements" page in the DevTools, and hover over every tag until you find what you want to edit.
  3. Change what you want to change
    This can be done by double clicking the tag. This enables you to edit the contents. Then, you can change the tag.
  4. Take a screenshot and send it to your friends!
